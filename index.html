(aaa) Result. Given an odd prime number $p$, we have exactly $p-1$ solutions to the equation $1+x^2=y^2$ with $x,y\in\Z/p$.

(aaa) Guidance. Use the factorization $y^2-x^2=(y-x)(y+x)$.

(aaa) Explanation. The bijection $(x,y)\mapsto (y-x,y+x)$ of $(\Z/p)^2$ allows us to count instead the number of solutions to $ab=1$ with $a,b\in\Z/p$, which is of course $p-1$.

(aad) Result. Given an odd prime number $p$, we have in $\Z/p$ a solution to the equation $x^2+1=0$ if and only if $p\equiv 1$ mod $4$.

(aad) Guidance. Apply (aaa).

(aad) Explanation. We apply (aaa). Non zero solutions to $1+x^2=y^2$ come in groups of four, namely when $(x,y)$ is a solution so is $(\pm x,\pm y)$. There are two solutions with $x=0$, and there either $2$ or $0$ solutions with $y=0$, depending on whether $1+x^2=0$ is soluble. Since there is a total of $p-1$ solutions, we have a solution to $1+x^2=0$ over $\Z/p$ iff $p\equiv 1$ mod $4$, as we wanted.

(aae) Result. Given an odd prime $p$, we have $\disp\(\sum_{k=0}^{p-1} e^{\frac{2\pi i}{p} k^2}\)^2=(-1)^{\frac{p-1}{2}}p$.

(aae) Guidance. Use the factorization $k^2-\ell^2=(k-\ell)(k+\ell)$. Also use (aad).

(aae) Explanation. Write $\xi=\disp\sum_{k=0}^{p-1} e^{\frac{2\pi i}{p} k^2}$. Then $\xi\bar{\xi}=\disp\sum_{k,\ell=0}^{p-1}e^{\frac{2\pi i}{p} (k^2-\ell^2)}$. As $k^2-\ell^2=(k-\ell)(k+\ell)$ and $(x,y)\mapsto (y-x,y+x)$ is a bijection of $\(\Z/p\)^2$, we have $\xi\bar{\xi}=\disp\sum_{k,\ell=0}^{p-1}e^{\frac{2\pi i}{p} (k\ell)}=p$. It remains to see that $\bar{\xi}=(-1)^{\frac{p-1}{2}}\xi$, and this is a consequence of (aad).

(aaf) Result. Given a harmonic $f:\R^n\to\C$ with $\disp\lim_{\abs{x}\to\infty}f(x)=\ell$, we have $f\equiv \ell$.

(aag) Given a monic polynomial $f(z)$ with complex coefficients, we have $f(z)=(z-\alpha_1)\dots(z-\alpha_d)$ for some (not necessarily distinct) $\alpha_1,\dots,\alpha_k\in\C$.

(aag) Guidance. Use (aaf).

(aag) Explanation. Assume $f$ does not vanish in $\C$. Then $g(z)=\dfrac{z^d}{f(z)}$ is an entire function, which equals zero at the origin, and approaches $1$ as $z$ approaches infinity. By (aaf), this is impossible.

(aab) Motivation. Recall that $\dfrac{f(x_0+h)-2f(x_0)+f(x_0-h)}{h^2}$ is an approximation of $f''(x_0)$.

(aab) Result. Given a continuous $f:[0,1]\to \R$, we have ${\displaystyle \lim_{h\to0}}\dfrac{f(t+h)-2f(t)+f(t-h)}{h^{2}}=0$ for all $t\in(0,1)$ iff $f(t)=at+b$ is affine.

(aab) Guidance. If instead the given limit was positive at $t$, then we could not have a maximum at $t$.

(aab) Explanation. By linearity, after removing an affine term we may assume $f(0)=f(1)=0$, so long as we yield $f\equiv 0$. Let $f_{\eps}(t)=f(t)-\eps t(1-t)$. Then $f_\eps(t)$ has a positive limit approximating the second derivative at each $t\in(0,1)$. Thus $f_\eps(t)$ cannot attain its maximum at any inner point, and so $f_\eps(t)\le\max(f_\eps(0),f_\eps(1))=0$. Letting $\eps$ tend to $0$ we find $f(t)\le 0$. Applying the same logic to $-f(t)$ yields $f\equiv 0$ as we wanted.

(aac) Result. Given a continuous function $f:\R\to\C\*$ we have $f(a+b)=f(a)f(b)$ for all $a,b\in\R$ iff $f(x)\equiv e^{\xi x}$ for some $\xi\in\C$.

(aac) Guidance. Any continuous function $f:\R\to\C$ can be expressed as $f(x)=r(x)e^{i\theta(x)}$ for some continuous radius and angle functions $r>0$, $\theta\in\R$.

(aac) Explanation. We express $f(x)=e^{g(x)}$ for a continuous $g:\R\to\C$. We see that $f(0)=1$ and so we may assume $g(0)=0$. We then have $g(a+b)=g(a)+g(b)+2\pi iq(a,b)$ for a continuous $q:\R^2\to\Z$. Of course, this means $q$ is constant and we have $g(a+b)=g(a)+g(b)$, and so $g(x)=\xi x$ for some $\xi\in\C$, which is of course what we wanted.

(aah) Result. There exists a family $\FF$ of negligible sets in $\R$ that forms a chain (namely $A,B\in\FF\implies A\sub B$ or $B\sub A$) whose union is not negligible.

(aah) Explanation. Otherwise, Zorn's lemma would imply the existence of a negligible subset of $\R$ maximal under inclusion, and such a set clearly cannot be.

(aai) Result. Given a positive integer $n$ and $\omega=e^{\frac{2\pi i}{n}}$, we have $\prod_{k\neq\ell \in[n]}(w^\ell-w^k)= (-1)^{n-1}n^n$.

(aaj) Result. Given a prime number $p$, a monic integral polynomial $f(t)\in\Z[t]$ and one of its complex roots $w$, we have that $f(w^p)$ is divisible by $p$ as an algebraic integer.

(aak) Result. The cyclotomic polynomial $\Phi_n(t)=\prod_{k\in[n],\gcd(k,n)=1}(t-e^{\frac{2\pi i}{n}k})\in\Z[t]$ is the minimal polynomial of $e^{\frac{2\pi i}{n}}$ over $\Q$.

(aak) Guidance. Use (aai) and (aaj).
